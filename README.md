## ğŸ® User Guide

### Navigation
- **Desktop**: Click navigation links or use keyboard shortcuts
- **Mobile**: Tap hamburger menu or swipe right to open navigation

### Keyboard Shortcuts
- `Alt + 1`: Home
- `Alt + 2`: Dashboard (logged in users)
- `Alt + 3`: Rewards (logged in users)
- `Alt + 4`: History (logged in users)
- `Alt + 5`: FAQ
- `Alt + 6`: Settings (logged in users)
- `Escape`: Close mobile menu

### Authentication
1. **Sign Up**: Create new account with email/phone
2. **Login**: Use existing credentials
3. **Social Login**: Google/Facebook integration (demo)

### Using the Dashboard
- View total points and recent activity
- Quick access to redemption options
- Monitor tier progress
- Track monthly statistics

### Redeeming Rewards
1. Browse rewards catalog
2. Filter by category (vouchers, discounts, freebies)
3. Click "Redeem Now" on desired reward
4. Confirm redemption in popup

### Viewing History
- Filter transactions by type and period
- View detailed transaction information
- Track points earned vs. redeemed
- Export transaction data
